Игра Well_played) начинается с заставки с названием игры и кнопкой "Старт"? которая подсвечивается при наведении на нее мыши.
После нажатия на кнопку игрока переносит в меню с уровнями, где пока открыт только первый. Нажав на кнопку с номером уровня игрок попадает на него.
На первых трех уровнях дверь, в которую должен зайти игрок уже открыта, на первых двух уровнях до двери нужно просто допрыгать, на третьем
уровне нужно создать для персонажа путь из блоков, появляющихся на месте нажатия мыши. На четвертом уровне нужно нажать все кнопки
в разных частях экрана, чтобы открыть дверь. На пятом уровне нужно решить загадку и записать кодовое слово "Скала", нажимая по буквам
в центре экрана(буквы меняются при нажатии по алфавиту). Пока реализовано 5 уровней. в меню можно выбрать любой из них.
Также прохождение игры можно остановить кнопкой паузы и продолжить прохождение по кнопке "Продолжить". При выходе из игры прогресс сохраняется,
и уже пройденные уровни будут открыти при следующем заходе в игру.
